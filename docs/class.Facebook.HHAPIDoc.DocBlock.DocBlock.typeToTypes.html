Convert a string type specifiction to a list of types

protected static function typeToTypes(
  ?string $type,
): vec<string>;

For example:

  • 'string' -> vec['string']
  • 'Foo|Bar' -> vec['Foo', 'Bar']
  • '[Foo|Bar]' -> vec['Foo', 'Bar']
